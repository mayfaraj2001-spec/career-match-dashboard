# CareerMatch Dashboard

## Goal
A web application for job seekers that helps users find jobs, save favorite jobs, and track application status.

## Problem
Job seekers need to browse many websites and manually track job applications.

## Target Users
Students, junior professionals, and job seekers in technology fields.

## Scope
Included:
- Job listings
- Search jobs
- Filter jobs
- Save favorite jobs
- Track application status
- Dashboard summary

Out of Scope:
- User authentication
- LinkedIn integration
- Backend database
- Resume upload

## Functional Requirements
1. Display jobs
2. Search jobs
3. Filter jobs
4. Save jobs
5. Track application status
6. Store data in LocalStorage

## Technical Stack
- HTML
- CSS
- JavaScript
- LocalStorage

## Data Source
Local jobs data stored in jobs.js

## Definition of Done
The application works without errors and all features are functional.
